# Floor is Liar

A 2D platformer game built in C using the Raylib library. It features multiple levels with troll traps and obstacles!

## How to Build and Run (Windows)

1. Make sure you have the MinGW GCC compiler installed.
2. Download the precompiled Raylib 5.0 for MinGW (`raylib-5.0_win32_mingw-w64`) and place it in the `raylib_precompiled` folder.
3. Run the provided build script from the terminal:
   ```
   .\build.bat
   ```
4. Once compiled, run the game:
   ```
   .\FloorIsLiar.exe
   ```

## Requirements
- GCC Compiler (MinGW)
- [Raylib](https://www.raylib.com/)
